Game Instructions
  1. Welcome to Bubble Burst! The objective of the game is to burst the bubbles conquering the screen!!!
  2. When the bubbles cover 75% of the screen the timer starts ticking and when it reaches 0, the game ends
  3. Rock bubbles require 5 clicks for demolition
  4. You can use liquid luck by pressing l which slows down rate of bubble generation... But use it cautiously as it can be used only twice throughout the game
  5. Half the bubbles get cleared upon clicking on the gaunlet that drops occasionally
  6. Game can be paused and resumed anytime by pressing p and r respectively
